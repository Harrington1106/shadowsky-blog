> **简介**：很多人购买云服务器后，除了简单的测试外往往闲置。其实，一台 24 小时在线的 Linux 服务器是极佳的个人数据中心和自动化平台。

## 一、 基础应用：搭建属于你的互联网家园

### 1. 个人博客 (WordPress / Hexo)
不再受制于第三方平台的审核与限制，拥有完全属于自己的数据。
*   **WordPress**: 动态博客霸主，插件丰富，适合想要功能强大、即使不懂代码的用户。推荐使用 `Docker` 一键部署。
*   **Hexo/Hugo**: 静态博客生成器，轻量、极速，适合喜欢 Markdown 写作的开发者。可以配合 GitHub Actions 自动部署到服务器 Nginx 上。

### 2. 私有云盘 (Nextcloud / Alist)
*   **Nextcloud**: 功能对标 Google Drive 的全功能私有云，支持多端同步、相册自动备份、在线协作办公。适合家庭数据中心。
*   **Alist**: 一个支持多种存储的目录文件列表程序。它可以挂载阿里云盘、百度网盘、Google Drive 等，将它们整合成一个可以在线播放视频的“本地硬盘”，彻底解决网盘限速和客户端臃肿的问题。

---

## 二、 进阶核心：Docker 容器化部署

强烈建议在服务器上使用 **Docker** 来管理所有应用。它能保证环境一致性，安装和卸载都不会污染宿主机系统。

### 1. 安装 Docker 环境
```bash
curl -fsSL https://get.docker.com | bash
```

### 2. 推荐部署的实用容器

*   **Nginx Proxy Manager**: 
    *   **用途**：可视化的反向代理管理工具。
    *   **价值**：让你轻松为一个 IP 绑定多个域名，并自动申请和续期 Let's Encrypt SSL 证书，无需手写复杂的 Nginx 配置文件。
    
*   **Uptime Kuma**:
    *   **用途**：极简的服务器监控工具。
    *   **价值**：监控你的博客、API 或其他服务是否在线，一旦宕机可以通过 Telegram、邮件等方式第一时间通知你。

*   **Bitwarden (Vaultwarden)**:
    *   **用途**：开源密码管理器。
    *   **价值**：将所有账号密码加密存储在自己服务器上，不再担心第三方服务（如 LastPass）的数据泄露风险。

## 三、 极客玩法：自动化与开发环境

### 1. 远程开发环境 (VS Code Remote)
通过 VS Code 的 `Remote - SSH` 插件连接服务器。
*   **场景**：使用 iPad 或性能较弱的笔记本，连接到高性能服务器进行编译和开发。
*   **优势**：代码运行在服务器上，不占用本地资源，且环境高度统一。

### 2. 自动化脚本 (Cron / Qinglong)
*   **Crontab**: Linux 自带的定时任务，适合简单的 Shell 脚本（如每天备份数据库）。
*   **青龙面板 (Qinglong)**: 强大的定时任务管理面板，支持 Python、JavaScript 等多种语言。常用于运行各种签到脚本、电商数据监控、自动抢购脚本等。

### 3. 网络环境搭建
如果你购买的是海外服务器（如美国、香港），它天然具备优质的国际网络互联能力。
*   **学习与科研**：通过搭建合规的代理服务，加速访问 GitHub、Docker Hub 以及各类学术资源。
*   *具体搭建方法请参考：**[阿里云轻量服务器搭建美国原生IP环境全流程指南](/posts/2025-12-08-alibaba-cloud-us-ip)**。*

## 四、 常用维护命令速查

掌握以下命令，随时掌控服务器健康状态：

*   **查看内存**: `free -h` (关注 `available` 字段)
*   **查看磁盘**: `df -h` (关注 `/` 根目录使用率)
*   **查看负载**: `htop` (直观的进程管理器，需先安装: `apt install htop`)
*   **查看端口**: `netstat -tunlp` (查看哪些端口正在被占用)
