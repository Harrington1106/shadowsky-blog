## 授人以渔：如何发现“潜力股”？

除了每天刷 GitHub Trending 榜单，作为开发者，我们还可以通过以下渠道第一时间获取前沿资讯：

1.  **关注“去中心化”榜单**：
    *   **Trendshift**: 相比 GitHub 官方榜单，它更能过滤掉刷 Star 的项目，专注于真实的开发者活跃度。
    *   **Hacker News**: 虽然界面复古，但上面的 *Show HN* 板块往往汇集了最具极客精神的硬核项目。

2.  **善用 Advanced Search (高级搜索)**：
    *   想找最近一周火起来的 Python 项目？
    *   试试这个搜索语法：`language:python created:>2025-12-01 sort:stars`。
    *   精准定位，不再大海捞针。

3.  **订阅优质 Newsletter**：
    *   **GitHub Explore Email**: 官方的个性化推荐，基于你 Star 过的项目及其开发者的动态。
    *   **Console.dev**: 每周推荐高质量的开发者工具，品味极佳。

## 📅 今日精选 (2025-12-11)

以下是为您筛选的今日热门项目，涵盖 AI Agent、Web 开发与生产力工具。

### 1. [browser-use/browser-use](https://github.com/browser-use/browser-use)
**让 AI 操控浏览器**
*   **语言**: Python
*   **简介**: 这是一个极具创新性的库，它允许 LLM（大语言模型）像人类一样直接操控浏览器。它可以自动完成网页点击、表单填写、数据抓取等任务。
*   **应用场景**: 自动化测试、网页爬虫、RPA (机器人流程自动化)。

### 2. [elizaOS/eliza](https://github.com/elizaOS/eliza)
**多平台自主 AI 代理**
*   **语言**: TypeScript
*   **简介**: 专为 Discord、Twitter 和 Telegram 设计的自主 AI Agent。它不仅能聊天，还能根据上下文执行复杂的社群管理任务，是目前最火的 DAO 治理工具之一。

### 3. [shadcn-ui/ui](https://github.com/shadcn-ui/ui)
**复制即用的 React 组件库**
*   **语言**: TypeScript
*   **简介**: 这不是一个传统的组件库（如 AntD 或 MUI）。Shadcn UI 提供的是“代码片段”。你直接把组件代码复制到你的项目中，完全拥有它，随意定制。
*   **评价**: 这种“非打包”的发布模式正在重塑前端开发的组件复用逻辑。

### 4. [Doriandarko/Maestro](https://github.com/Doriandarko/Maestro)
**轻量级 AI 编排框架**
*   **语言**: Python
*   **简介**: 一个极其轻量的框架，用于编排多个 AI 模型协同工作。它没有复杂的依赖，主打“简单、可控”，非常适合想要快速构建 Multi-Agent 应用的开发者。

### 5. [ComposioHQ/composio](https://github.com/ComposioHQ/composio)
**AI Agent 的工具箱**
*   **语言**: Python / JavaScript
*   **简介**: 为你的 AI Agent 提供超过 100 种集成工具（GitHub, Notion, Linear, Slack 等）。通过 Composio，你的 AI 不再是只会聊天的机器人，而是能真正干活的助手。
