> **警告**：Hysteria 2 协议较为激进，虽然能显著提升速度，但在部分地区可能会被运营商短时阻断（表现为断网几分钟）。建议配合 Shadowsocks 作为备用。

## 为什么你需要 Hysteria 2？

如果你的 VPS（如 RackNerd, CloudCone 等廉价商家）出现以下症状：
1.  **白天**：看 4K 视频毫无压力。
2.  **晚上 (20:00-23:00)**：看 720p 都卡顿，丢包率超过 10%。

这是因为物理线路（163骨干网）堵车了。传统的 TCP 协议（Shadowsocks, VMess）会因为丢包而不断减速等待重传。
而 **Hysteria 2 (歇斯底里)** 基于 UDP，它不管路堵不堵，强行塞入数据包，从而在拥堵路段抢占带宽。

## 部署步骤 (全自动脚本)

我们使用开源社区维护的一键脚本，支持**自动生成自签证书**（不需要你有域名），非常适合新手。

### 1. 连接服务器
使用 SSH 客户端连接到你的 VPS。

### 2. 执行安装命令
复制以下命令并在终端粘贴回车：

```bash
wget -N --no-check-certificate https://raw.githubusercontent.com/Misaka-blog/hysteria-install/main/hy2/hysteria.sh && bash hysteria.sh
```

### 3. 按提示操作
脚本运行后，会出现中文菜单，按照以下逻辑选择：
1.  选择 **[1] 安装 Hysteria 2**。
2.  **证书类型**：选择 **自签证书** (通常是选项 1 或 2，寻找 "Self-signed" 或 "自签" 字样)。
    *   *注意：如果你有域名，也可以选申请证书，但自签最简单，无需任何准备。*
3.  **端口**：建议输入 `40000` 到 `60000` 之间的数字，例如 `54321`。
    *   *注意：不要和已有的 Shadowsocks 端口冲突。*
4.  **密码**：设置一个复杂的密码，或者直接回车使用随机密码。
5.  **伪装站点**：直接回车默认（bing.com）即可。

### 4. 获取配置
安装完成后，屏幕上会显示红色的配置信息，包括：
*   IP
*   端口
*   密码
*   SNI (伪装域名)
*   指纹 (Fingerprint) - **自签证书模式下非常重要**

还可以选择 **导出 v2rayN 配置文件**，或者直接复制 `hysteria2://...` 开头的链接。

## 客户端配置指南

### Windows (v2rayN)
1.  下载最新版 **v2rayN** (版本 >= 6.0)。
2.  复制服务器输出的 `hysteria2://` 链接。
3.  在 v2rayN 中按 `Ctrl + V` 导入，或者点击 **服务器** -> **从剪贴板导入**。
4.  **关键设置** (参考下图你的界面)：
    *   双击打开节点配置。
    *   **SNI**：在中间位置，默认通常是 `www.bing.com` (或者是你自定义的域名)。
    *   **跳过证书验证 (allowInsecure)**：在倒数第二行，必须设为 **true** (因为我们用的是自签证书)。


### Android (NekoBox)
1.  安装 **NekoBox for Android**。
2.  复制 `hysteria2://` 链接，打开 App 会自动识别导入。
3.  如果手动配置，务必勾选 **"允许不安全连接"**。

### iOS (Shadowrocket)
1.  确保小火箭是最新版。
2.  添加节点 -> 类型选择 **Hysteria2**。
3.  填入 IP、端口、密码。
4.  **允许不安全**：开启。
5.  **SNI**：填入 www.bing.com (或者你设置的伪装域名)。

## 常见问题

**Q: 速度还是慢？**
A: 检查你的宽带运营商是否对 UDP 进行了限速（QoS）。部分地区（如广东电信）对 UDP 非常不友好。

**Q: 经常断流？**
A: Hysteria 2 抢占带宽太猛，可能被运营商检测到并临时阻断。解决方法：
1.  切换回 Shadowsocks 用一会儿。
2.  在客户端设置中降低 **上传/下载带宽** 限制（例如限制在 50Mbps），不要跑满，低调一点。

**Q: 能和 Shadowsocks 共存吗？**
A: **完美共存**。只要端口不同（例如 SS 用 10000，Hy2 用 20000），它们互不影响。建议同时保留，互为备胎。
