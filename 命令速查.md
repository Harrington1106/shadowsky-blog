# 博客常用命令

> 本地路径：`D:\Projects\shadowsky-blog`
> 服务器：`shadowsky` (47.118.28.27)
> 网站目录：`/www/wwwroot/47.118.28.27/`

---

## 拉取到本地

```powershell
# 在 D:\Projects\shadowsky-blog 下打开 PowerShell
git pull
```

## 部署到服务器

```powershell
git add -A
git commit -m "你的改动说明"
git push
```

推送后服务器自动更新（git hook `updateInstead`），即刷新浏览器（Ctrl+Shift+R）即可看到。

## 单文件快速部署（不经过 git）

```bash
# 在 WSL 里
scp blog.html shadowsky:/www/wwwroot/47.118.28.27/
```

## SSH 进服务器

```bash
ssh shadowsky
```

## 在服务器上查看状态

```bash
ssh shadowsky "cd /www/wwwroot/47.118.28.27 && git status"
ssh shadowsky "cd /www/wwwroot/47.118.28.27 && git log --oneline -5"
```

## 服务器文件管理

```bash
# 列出网站目录
ssh shadowsky "ls -la /www/wwwroot/47.118.28.27/"

# 实时看 nginx 日志
ssh shadowsky "tail -f /www/logs/shadowquake.top.error.log"

# 重启 Node 后端
ssh shadowsky "pm2 restart shadowsky-admin"
```

## WSL ↔ Windows 路径对照

| Windows | WSL |
|---------|-----|
| `D:\Projects\shadowsky-blog` | `/mnt/d/Projects/shadowsky-blog` |

WSL 终端里用 `/mnt/d/...`，Windows PowerShell 里直接用 `D:\...`。
