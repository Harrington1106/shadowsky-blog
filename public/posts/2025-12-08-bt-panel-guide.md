---

## 建站：从地狱到天堂

这是宝塔最核心的功能。没有宝塔之前，搭建一个网站你需要：

1. SSH 连上服务器（先学会用 Putty 或终端）
2. `apt install nginx`（等半天）
3. `apt install mysql-server`（配置密码、权限、字符集）
4. `apt install php-fpm php-mysql php-curl ...`（十几个扩展，少一个都报错）
5. 写 Nginx 配置文件（一个分号错了就 502）
6. 申请 SSL 证书（certbot 命令行，一周后过期了你都不知道）

**有宝塔之后：**

1. 登录宝塔网页面板
2. 点"软件商店" → 一键安装 LNMP（Linux + Nginx + MySQL + PHP）
3. 等 5 分钟，环境全自动配好
4. 点"网站" → "添加站点" → 填域名 → 确定
5. 点"SSL" → "Let's Encrypt" → "申请" → 证书自动装上，到期自动续

整个过程不超过 10 分钟，而且**不会报错**。

## 文件管理：告别 FTP 工具

宝塔自带了一个在线文件管理器，功能非常完善：

- **拖拽上传**：直接把文件从桌面拖进浏览器
- **在线编辑**：双击文件就能改代码，支持语法高亮
- **压缩解压**：右键就能解压 ZIP/TAR，不用命令行
- **权限管理**：777 / 755 不再需要背数字，点复选框就行
- **远程下载**：粘贴 URL，服务器直接下载文件

> 💡 **实用场景**：网站搬家时，把旧站文件打包成 ZIP 上传到宝塔，右键解压，全程 30 秒。

## 计划任务：让服务器自己干活

你可以设置定时任务，服务器会在指定时间自动执行：

### 自动备份（最重要！）

```
每天凌晨 3:00 → 备份网站代码 + 数据库 → 上传到阿里云 OSS / 腾讯云 COS / Google Drive
```

设置一次，永久生效。哪天服务器挂了，一键恢复。

### 其他自动化

| 任务 | 用途 |
|------|------|
| **日志切割** | 防止 nginx 日志撑爆磁盘 |
| **释放内存** | 每天凌晨重启 PHP/MySQL，清理内存碎片 |
| **证书续期** | SSL 证书到期前自动续（其实宝塔默认就会） |
| **URL 监控** | 定时检查网站是否在线，挂了发邮件通知 |

## 软件商店：点一下就能装

宝塔内置了一个应用商店，装软件跟装手机 App 一样：

| 软件 | 干什么用 |
|------|----------|
| **Redis** | 缓存加速，网站快 3 倍 |
| **Docker 管理器** | 图形化管 Docker 容器 |
| **PM2 管理器** | 跑 Node.js 项目必备 |
| **Fail2ban** | 自动封禁暴力破解 IP |
| **Nginx WAF** | Web 应用防火墙（部分付费） |

> ⚠️ **注意**：宝塔的应用商店主要面向简单场景。如果你需要复杂的 Docker 编排，还是建议用 Portainer 或直接用命令行。

## 监控与安全

打开宝塔首页，一个炫酷的仪表盘：

- CPU 使用率（实时波形图）
- 内存占用
- 磁盘读写速度
- 网络流量
- 负载状态

一眼就能看出服务器有没有异常。内存飙红了？点进去看哪个进程在吃资源，一键杀掉。

安全方面：

- **系统防火墙**：可视化配置端口规则，屏蔽恶意 IP
- **SSH 管理**：改端口、禁 root 登录、设置密钥登录
- **面板 SSL**：给宝塔面板本身也配上 HTTPS

| 需求 | 操作 |

## 总结

宝塔面板不是万能药，但它是**最好的入门工具**。它让你可以：
- 不用学 Linux 命令就能建站
- 不用记配置文件语法就能配环境
- 不用担心忘记备份

等你玩熟了，自然会想学命令行做更精细的控制。但在那之前，宝塔让你的服务器之旅从"劝退"变成了"真香"。
