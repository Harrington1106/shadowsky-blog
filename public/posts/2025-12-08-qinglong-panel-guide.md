> **TL;DR**：青龙面板 = 自动化管家。帮你每天自动签到领京豆、自动抢券、自动完成各种 APP 日常任务。原理就是定时执行脚本，脚本帮你"打工"。

---

## 薅羊毛（最出圈的玩法）

青龙面板最广为人知的用途，就是帮你自动完成各种 APP 的每日任务：

| 平台 | 能自动干什么 | 收益 |
|------|-------------|------|
| **京东** | 签到、浇水、喂鸡、领京豆、抢券 | 一年几百块京豆 |
| **饿了么/美团** | 领红包、吃货豆 | 点外卖便宜 |
| **Bilibili** | 登录、看视频、投币、分享 | 快速升级 Lv6 |
| **阿里云盘** | 每日签到领空间 | 扩容免费容量 |
| **百度网盘** | 自动签到 | 领 SVIP 体验 |

> ⚠️ **原理说明**：你需要把自己的账号 Cookie 填进脚本。脚本到点了就模拟你的操作去完成这些任务。等于雇了个不要工资的机器人。

## 自动签到（不只是薅羊毛）

除了电商平台，青龙还能处理各种签到需求：

- **机场签到**：自动签到获取流量
- **论坛/贴吧签到**：保持账号活跃度
- **GitHub 签到**：维持 Contribution 绿格子（虽然没啥实际意义）

想加新任务？去 GitHub 搜 "青龙面板 脚本"，有海量的开源脚本库。复制粘贴到青龙里，设好定时，开跑。

## 依赖管理：比 Crontab 好在哪里

传统的 Linux Crontab 也能跑定时脚本，但有个致命问题：**装依赖太痛苦**。

脚本需要 `axios`、`requests`、`pandas` 等各种库。用 Crontab，你得手动一个一个 `pip install` 或 `npm install`，换台服务器又得重来一遍。

青龙面板有图形化的依赖管理：
- 打开依赖管理页面
- 搜你要的库名
- 点安装

Node.js 的、Python 的、Linux 系统包，全都一键搞定。

> 💡 **如果你写自己的脚本**：青龙支持 Python3、JavaScript、TypeScript、Shell 四种语言，几乎覆盖所有常见自动化需求。

## 风险提示（认真看）

### 账号安全

Cookie 约等于账号密码。你把它交给脚本，就等于把钥匙给了别人。

**如何降低风险：**
- 只用 **GitHub Stars > 500** 的知名脚本库
- 阅读脚本源码，确认没有向陌生地址发送数据
- 用**小号**跑高风险脚本
- 定期更换 Cookie

### 风控封号

频繁的自动化操作可能被平台判定为机器人，导致：
- 领不到券（"活动太火爆"）
- 账号被降权
- 严重情况直接封号

**建议**：不要 24 小时不间断跑，设置合理的执行间隔（比如每天跑一次）。

## 快速上手

```bash
# Docker 一键安装青龙面板
docker run -dit \
  --name qinglong \
  --restart always \
  -p 5700:5700 \
  -v /opt/qinglong/data:/ql/data \
  whyour/qinglong:latest
```

装好后访问 `http://你的IP:5700`，按引导完成初始化，然后去 GitHub 搜 "青龙面板 京东脚本" 跟着教程配置就行了。

## 总结

青龙面板的真正身份是**通用定时任务平台**，薅羊毛只是其中一个应用场景。你还可以用它：
- 定时抓取数据（爬虫）
- 定时备份文件
- 定时检查网站状态
- 定时清理服务器垃圾

只要你能用脚本描述的任务，青龙就能帮你自动化。
